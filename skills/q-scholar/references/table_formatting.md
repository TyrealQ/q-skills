# APA 7th Edition Table Formatting

## Basic Table Structure

Tables in APA format have four components: number, title, body, and notes.

```
Table 1

Descriptive Title Explaining Table Contents

| Column Header | Column Header | Column Header |
|---------------|---------------|---------------|
| Row data      | Row data      | Row data      |
| Row data      | Row data      | Row data      |

Note. Explanation of abbreviations, symbols, or context.
```

## Formatting Rules

### Table Number
- Plain text (not italicized, not bold)
- On its own line above the title
- Sequential numbering: Table 1, Table 2, Table 3

### Table Title
- Italicized in final manuscript (plain in markdown drafts)
- Title case capitalization
- Brief but descriptive
- On its own line below the number

### Table Body
- No vertical lines (horizontal rules only)
- Left-align text columns
- Right-align numeric columns
- Use consistent decimal places within columns

### Table Notes
- Begin with "Note." in italics followed by period
- General notes first, then specific notes, then probability notes
- Abbreviations defined in notes

## Example Tables by Type

### Frequency Distribution Table

```
Table 1

Distribution of Posts by Stakeholder Category

| Stakeholder | Posts | Percentage |
|-------------|------:|----------:|
| Fan | 66,501 | 46.12 |
| Media | 27,464 | 19.04 |
| Content creator | 13,844 | 9.60 |
| Player | 11,510 | 7.98 |
| Other | 24,887 | 17.26 |
| Total | 144,206 | 100.00 |

Note. Percentages may not sum to 100 due to rounding.
```

## Column Alignment Guidelines

| Data Type | Alignment | Example |
|-----------|-----------|---------|
| Text/labels | Left | Category names |
| Whole numbers | Right | Counts, frequencies |
| Decimals | Right (align on decimal) | Percentages, means |
| Mixed text/numbers | Left | Codes with labels |

## Notes Types

### General Notes
Explain the table as a whole, define abbreviations.
> Note. Data collected August 2022 through August 2024. EWC = Esports World Cup.

### Specific Notes
Explain specific cells or columns using superscript letters.
> Note. ^a Category includes both subcategories A1 and A2.

### Probability Notes
Report significance levels.
> *p < .05. **p < .01. ***p < .001.

## Common Mistakes to Avoid

1. Using vertical lines in tables
2. Forgetting to reference tables in text before they appear
3. Inconsistent decimal places within columns
4. Titles that are too vague ("Results" vs. "Distribution of Topics by Legitimacy Type")
5. Missing notes explaining abbreviations
6. Bold or italic emphasis within table cells
